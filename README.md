JavaScript Fundamentals Quiz Example
------
Hello, this is an example quiz I made with HTML, CSS, and JavaScript for an assignment at Coding Temple. The quiz features 4 questions that covers basic concepts in JavaScript. Upon loading the page, the user is greeted with a small message and a start button.
Clicking the start button will begin the quiz. Correct answers will be highlighted green upon clicking and incorrect answers will be highlighted in red. At the end of the quiz, the user will be the shown how many questions out of 4 they have correct and will be given an option to restart the quiz.
